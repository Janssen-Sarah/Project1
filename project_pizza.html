Eerste project

Mijn allereerste project betekende meteen een sprong in het diepe. Als nieuwkomer in de IT-wereld, en met mijn instap middenin het academiejaar, was het een behoorlijke uitdaging om mijn weg te vinden. De onzekerheid en het onbekende maakte het er niet makkelijker op. Toch was ik vastberaden om door te zetten en er volop voor te gaan. Naarmate ik meer en meer oefende, vond ik uiteindelijk de weg om een mooi eerste project neer te zetten. Ik maakte een SQL-database voor een fictief pizzabedrijf. Hieronder kan u alvast meer lezen over dit project.

Kennismaking met SQL

Dit project betreft het omzetten van een bestaand ERD-schema naar een SQL-database voor een fictief pizzabedrijf. De opdracht omvat het vertalen van het ERD-diagram naar een SQL-script voor het aanmaken van de tabellen, waarbij alle elementen in het datamodel worden aangegeven, zoals het datatype, FK/PK en NULL/NOT NULL constraints. Het script dropt eerst alle tabellen en maakt ze daarna opnieuw aan, zodat het meerdere keren kan worden uitgevoerd. Ik maakte ook een schema aan om de tabellen in te plaatsen. Verder moesten verschillende records worden ingevuld voor elke tabel, inclusief de associatie-entiteiten. Het INSERT-INTO-statement wordt gebruikt om deze records toe te voegen. Dit project is een eerste praktijkervaring in het gebruik van SQL Server Management Studio en het omzetten van een ERD-schema naar een SQL-database.
